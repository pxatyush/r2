# dyino Assets Guide

## sounds/
Place your .mp3 or .ogg sound files here.

### Naming Convention:
- Simple: `rain.mp3` → appears as "Rain" in the "Natural" category
- Categorized: `binaural_alpha.mp3` → appears as "Alpha" in the "Binaural" category

### Built-in sound names (add these files to enable them):
Natural category:
  rain.mp3, forest.mp3, ocean.mp3, river.mp3, wind.mp3, birds.mp3, thunder.mp3, beach.mp3

Binaural category:
  binaural_alpha.mp3, binaural_beta.mp3, binaural_theta.mp3, binaural_delta.mp3, binaural_gamma.mp3, binaural_432hz.mp3

Ambient category:
  ambient_fire.mp3, ambient_whitenoise.mp3, ambient_cafe.mp3, ambient_zenbowl.mp3, ambient_space.mp3, ambient_night.mp3

### Button click sound:
  click.mp3  (played when pressing buttons if Button Sound Effects is ON)

## radio/
- config.txt → Radio station list (see format inside file)

## fonts/
Place .ttf or .otf font files here. They will appear in Settings > Font Style.

## icon.png
Replace the launcher icon here. Must be a square PNG (512x512 recommended).
